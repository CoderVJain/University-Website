# Edusity - University Website

Edusity is a modern, responsive, and visually appealing University Website built with React. It showcases various educational programs, campus life, student testimonials, and provides a way for prospective students to contact the administration.

**Live Demo:** [https://university-new.vercel.app/](https://university-new.vercel.app/)

## 🌟 Features

- **Dynamic Hero Section:** Engaging landing page with a clear call to action.
- **Smooth Scrolling Navigation:** Utilizes `react-scroll` for a seamless single-page navigation experience.
- **Programs & Courses:** Display of various degree programs and courses offered.
- **About Us:** Information about the university, its history, and values.
- **Campus Gallery:** Visual representation of the campus facilities and student life.
- **Student Testimonials:** Feedback and reviews from current and past students.
- **Contact Form:** Easy to use contact section for inquiries.
- **Fully Responsive:** Optimized for desktops, tablets, and mobile devices.

## 🛠️ Technologies Used

- **Frontend:** React.js, HTML5, Vanilla CSS
- **Navigation:** `react-scroll`
- **Deployment:** Vercel

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js installed on your machine
- npm or yarn package manager

### Installation

1. **Navigate to the project directory:**
   ```sh
   cd University-Website
   ```

2. **Install the dependencies:**
   ```sh
   npm install
   ```

3. **Start the development server:**
   ```sh
   npm start
   ```

4. **Open your browser:**
   Open [http://localhost:3000](http://localhost:3000) to view the application in your browser. The page will automatically reload when you make code changes.
